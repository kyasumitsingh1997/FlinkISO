Simple JQuery plugin for all your countdown requirements.

Really very easy to implement and with all important features.

Main features

  • Multiple instances on the same page
  • No JS init is needed
  • Two entry modes, one for very simples countdowns, and other for cases where more CSS is needed.
  • Timer Mode
  • Pluralization
  • Time zones
  • Configurable Zero Padding

Simple Examples

(days) p_days, (hours) p_hours, and (minutes) p_minutes left!
(days) p_days, (hours) p_hours, and (minutes) p_minutes left!

Timer Mode
(example starts with 3 days, 2 hours and 10 minutes)

(days) p_days, (hours) p_hours, (minutes) p_minutes and (seconds) p_seconds left!

Timer Mode with configurable Zero Padding

(days) p_days - (hours):(minutes):(seconds) left!
